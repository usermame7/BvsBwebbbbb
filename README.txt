
Bulls vs Bears - Complete Trading Engine

Included:
- Smart Contract: USDT-based binary options (contracts/BvsB_USDT.sol)
- Backend server: Express + Ethers.js logic (backend/server.js)
- Wallet logic: Generate per-user deposit addresses (wallet.js)
- Mock DB schema
- Auto/manual resolution logic
- CoinGecko price API

Use: Install node modules, configure .env, and run: `node server.js`
